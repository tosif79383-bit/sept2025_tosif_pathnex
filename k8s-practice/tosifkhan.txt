# Artifacts and Archiving

## ðŸ”¹ Jenkins Pipeline â€” Archive Artifacts
You will learn how to **save build artifacts for future use**.

pipeline {
    agent any
    environment {
        INSTITUTE_NAME = "tosif"
    }
    stages {
        stage('Checkout') {
            steps {
                git url: 'https://github.com/tosif/sample-java-app.git'
            }
        }
        stage('Build') {
            steps {
                sh 'mvn clean package'
            }
        }
        stage('Archive') {
            steps {
                archiveArtifacts artifacts: 'target/*.jar', fingerprint: true
            }
        }
    }
}

## ðŸ”¹ GitLab CI â€” Artifacts
You will learn how to **store artifacts in GitLab CI**.

stages:
  - build

build:
  stage: build
  image: maven:3.8.1-jdk-17
  script:
    - git clone https://github.com/tosif/sample-java-app.git
    - cd sample-java-app
    - mvn clean package
  artifacts:
    paths:
      - target/*.jar
    expire_in: 1 week
